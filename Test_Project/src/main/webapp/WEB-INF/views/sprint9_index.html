Sprint 9 : API REST

Retour automatique de JSON avec @RestAPI

Fonctionnalités du Sprint 9

  • @RestAPI : Annotation pour marquer les méthodes qui retournent du JSON
  • Objet → JSON : Conversion automatique des objets en JSON
  • Liste → JSON : Conversion avec compteur (count)
  • ModelView → JSON : Extraction des données du ModelView
  • Format standardisé : Réponses avec status, code, data/result

Test 1 : Objet simple

URL : /api/employee/1

Retourne un objet Employee converti en JSON.

@RestAPI @GetUrl("/api/employee/{id}") public Employee getEmployee(Integer id) { return employee; }

Test 2 : Liste d'objets

URL : /api/employees

Retourne une liste d'employés avec compteur.

@RestAPI @GetUrl("/api/employees") public List<Employee> getEmployees() { return employees; }

Test 3 : String

URL : /api/message

Retourne un message texte en JSON.

@RestAPI @GetUrl("/api/message") public String getMessage() { return "Hello from REST API!"; }

Test 4 : Integer

URL : /api/count

Retourne un nombre en JSON.

@RestAPI @GetUrl("/api/count") public Integer getCount() { return 42; }

Test 5 : ModelView → JSON

URL : /api/data

Les données du ModelView sont extraites et converties en JSON.

@RestAPI @GetUrl("/api/data") public ModelView getData() { ModelView mv = new ModelView(); mv.addObject("name", "Jean"); return mv; }
